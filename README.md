# Datacenter-manager

**The Professional Infrastructure & Monitoring Platform** *A powerful, enterprise-grade fork of the RackTables project.*

---

## 🚀 Overview
**Datacenter-manager** is a specialized fork of RackTables designed for high-load data centers, Network Operation Centers (NOC), and complex IT infrastructures. While preserving the core inventory logic of the original project, Datacenter-manager introduces a massive suite of features focused on **automation, real-time monitoring, and team collaboration.**

This project transforms a static asset database into a dynamic infrastructure management platform.

---

## ✨ Key Features (Beyond RackTables)

### 1. Advanced Monitoring Suite
* **IPMI Management:** Real-time power status tracking and remote power control (ON/OFF/Reset) directly from the UI.
* **SNMP Traffic Analysis:** High-performance polling of switch ports with 95th percentile billing support.
* **ICMP Availability:** Scalable, sharded ICMP polling for global host reachability.
* **Multi-tier Data Storage:** Optimized SQL schemas for RAW, hourly, and daily data aggregation.

### 2. Business & Billing Tools
* **Traffic Billing Reports:** Generate precise reports based on port usage for client invoicing.
* **Interactive Visuals:** Dynamic traffic graphs powered by Chart.js with customizable timeframes.

### 3. Integrated Collaboration Environment
* **Embedded Wiki:** A full-featured documentation system with file storage for SOPs and manuals.
* **Task Management:** A built-in ticketing/task system linked to infrastructure objects.
* **Telegram Integration:** Real-time incident and task notifications via Telegram Bot API.

### 4. Enterprise Infrastructure
* **No-Java KVM Console:** HTML5-based remote console access (removing legacy Java dependencies).
* **MAAS Integration:** Seamless connection with Canonical MAAS for bare-metal provisioning.
* **Server-Split Architecture:** Ready-to-use architecture for separating UI, Database, and Polling agents for maximum performance.
